# Обслуживание клиентов оператора сотовой связи

Консольная программа по алгоритмам и структурам данных по предметной области: обслуживание клиентов оператора сотовой связи


## Состоит из: 

- `mobile_operator_system.py` - запуск программы
- `cli.py` - консольное меню и ввод данных
- `system.py` - основная логика работы с клиентами, SIM-картами и выдачами
- `models.py` - классы клиента, SIM-карты и записи выдачи
- `structures.py` - структуры данных: АВЛ-дерево, хеш-таблица и однонаправленный список
- `search.py` - прямой поиск подстроки
- `fixtures.py` - пробные данные для быстрой проверки

## Используемые структуры и алгоритмы

Клиенты хранятся в АВЛ-дереве по номеру паспорта. Это дает быстрый поиск, добавление и удаление данных.

SIM-карты хранятся в хеш-таблице с двойным хешированием. При заполнении таблица расширяется.

Записи выдачи SIM-карт лежат в однонаправленном списке, который поддерживается отсортированным по номеру SIM-карты.

Для поиска клиентов по фрагменту ФИО или адреса используется прямой поиск подстроки.

## Запуск

```bash
python mobile_operator_system.py
```

При старте программа спросит, нужно ли добавить тестовые данные:

```text
Добавить тестовые данные? (1 - да, 0 - нет):
```
При выбора `1`, будут добавлены несколько клиентов, SIM-карт и готовых записей выдачи.
