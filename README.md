# Система учета SIM-карт мобильного оператора

Небольшая консольная программа для курсовой работы по алгоритмам и структурам данных.
В системе можно хранить клиентов, SIM-карты и записи о выдаче SIM-карт клиентам.

## Что внутри

- `mobile_operator_system.py` - точка запуска программы.
- `cli.py` - консольное меню и ввод данных.
- `system.py` - основная логика работы с клиентами, SIM-картами и выдачами.
- `models.py` - классы клиента, SIM-карты и записи выдачи.
- `structures.py` - собственные структуры данных: АВЛ-дерево, хеш-таблица и однонаправленный список.
- `search.py` - прямой поиск подстроки.
- `fixtures.py` - пробные данные для быстрой проверки.

## Используемые структуры и алгоритмы

Клиенты хранятся в АВЛ-дереве по номеру паспорта. Это дает быстрый поиск, добавление и удаление с автоматической балансировкой дерева.

SIM-карты хранятся в хеш-таблице с двойным хешированием. При заполнении таблица расширяется и перестраивается.

Записи выдачи SIM-карт лежат в однонаправленном списке, который поддерживается отсортированным по номеру SIM-карты.

Для поиска клиентов по фрагменту ФИО или адреса используется простой прямой поиск подстроки.

## Запуск

```bash
python mobile_operator_system.py
```

При старте программа спросит, нужно ли добавить тестовые данные:

```text
Добавить тестовые данные? (1 - да, 0 - нет):
```

Если выбрать `1`, будут добавлены несколько клиентов, SIM-карт и готовых записей выдачи. Это удобно для проверки поиска, списков и возврата SIM-карты.
